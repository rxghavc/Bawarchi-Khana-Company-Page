# Bawarchi Khana - Catering Landing Page

A modern, responsive catering landing page that highlights Bawarchi Khana’s services, specials, delivery options, FAQs, and customer reviews. The site was built on a responsive UI and is user-friendly. Highlight images of various dishes fill the page to give the user a glimpse of the rich dishes available. The website is available at: https://bawarchikhaana.com/

## Key Features

- Company Highlights
- About the Cook
- Menu Specials
- Google/Instagram Reviews
- FAQs
- Delivery Options
- Contact the Cook

## 🛠️ Tech Stack

- Next.js (App Router) + React
- TypeScript
- Tailwind CSS + PostCSS
- Serverless API routes (Next.js route handlers)

## Prerequisites

- Node.js
- A Gmail account with an App Password (for the contact form email delivery)

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/rxghavc/Bawarchi-Khana-Company-Page.git
cd landing-page
```

### 2. Install Dependencies

```bash
npm install
# or
yarn install
```

### 3. Set Up Environment Variables

Create a `.env` file in the project root and add your Gmail app credentials:

```
GMAIL_USER=your-email@gmail.com
GMAIL_APP_PASSWORD=your_app_password
```

### 4. Run the Development Server

```bash
npm run dev
# or
yarn dev
```

The app will be available at the URL shown by Next.js (commonly `http://localhost:3000`).

### 5. Build for Production

```bash
npm run build
# or
yarn build
```
